# 98 Research

- Deterministic trend labels only need recent ordered values plus a fixed delta threshold.
- Volatility can be expressed as normalized movement between adjacent values with low/high bands.
- Recurrence can be grounded in recent regression counts and repeated top-driver failure types.
- Dataset health is best expressed as a trend label plus a coarse health label (`improving`,
  `stable`, `volatile`, `stale`) over versioned pack evaluation runs.
